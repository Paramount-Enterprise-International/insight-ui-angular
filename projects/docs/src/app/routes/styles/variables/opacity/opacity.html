Opacity

Opacity variables for creating consistent transparency effects across the application. These variables provide predefined opacity levels for different visual states.


Opacity Variables Reference

Variable Name CSS Value Description
--i-opacity-transparent 0px Transparent: Completely invisible elements. Used for hidden content, collapsed elements, or transparent overlays.
--i-opacity-disabled 0.4000000059604645px Disabled State: Semi-transparent appearance for disabled UI elements. Indicates non-interactive states while maintaining content visibility.
--i-opacity-muted 0.699999988079071px Muted/Secondary: Reduced opacity for secondary content, placeholder text, or less important visual elements.
--i-opacity-solid 1px Solid/Default: Full opacity for primary content, active elements, and main visual components. Default state for most UI.
