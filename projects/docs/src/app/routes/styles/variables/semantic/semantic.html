Input Border Variables (Semantic)

Semantic input border variables that describe purpose and function rather than just appearance. These variables define the visual styling of form inputs based on their interactive states.


Semantic Input Border Variables Reference

Variable Name CSS Value Usage & Description
--i-input-border-color var(--i-color-gray-200) Default Input Border: Standard border color for form inputs in their resting state. Semantic naming clearly indicates this is for input elements specifically.
--i-input-border-focus-color var(--i-color-blue-500) Focused Input Border: Border color for form inputs when they have user focus. The name describes the interactive state rather than just the color.
--i-input-border 1px solid var(--i-input-border-color) Complete Input Border: Full border definition for all input elements. Semantic because it's tied to input components, not generic borders.
