We're using Fontawesome Free for icons, there are several icons that being mapped to create shorter name.


Properties :

Property Data Type Required Default Value Remark
icon IIconName|string Optional false
size IIconSize Optional 'md' '2xs', 'xs', 'sm', 'md', 'lg', 'xl', '2xl'

IIconName

The name of the icon, mapped Fontawesome Free icon class


IIconSize

There are 6 icon sizes :

  • 2xs
  • xs
  • sm
  • md
  • lg
  • xl
  • 2xl
