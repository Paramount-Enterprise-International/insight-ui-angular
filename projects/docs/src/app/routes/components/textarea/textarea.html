Textarea

i-textarea & i-fc-textarea

Textarea component with form control integration, validation support, and customizable error messages.


Components

  • i-textarea - Basic textarea component implementing ControlValueAccessor
  • i-fc-textarea - Form control wrapper with label and error message

Properties (i-textarea)

Property Type Default Description
placeholder string '' Placeholder text for the textarea
readonly boolean false Makes the textarea read-only
rows number 3 Number of visible text lines
invalid boolean false Visual invalid state indicator
value string | null null Current value of the textarea
disabled boolean false Disables the textarea input

Properties (i-fc-textarea)

Property Type Description
label string Label text displayed above textarea
placeholder string Placeholder text for the textarea
readonly boolean Makes the textarea read-only
rows number Number of visible text lines (default: 3)
errorMessage IFormControlErrorMessage Custom error message templates
value string | null Current value (non-reactive binding)

Form Integration Properties

Property Type Description
formControlName string Reactive forms control name
