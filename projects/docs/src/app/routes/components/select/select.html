Select

i-select & i-fc-select

Dropdown select component with search features, custom templates, and form control integration.


Components

  • i-select - Basic select component
  • i-fc-select - Form control wrapper with label and error message
  • [iSelectOption] - Directive for custom option templates

Properties (i-select)

Property Type Default Description
options T[] | null [] Array of options
options$ Observable<T[]> | null null Observable options for async loading
placeholder string '' Input placeholder text
displayWith (row: T) => string | string (row) => String(row) Custom display text mapping
panelPosition ISelectPanelPosition 'bottom left' Dropdown panel position (top/bottom/left/right)
filterDelay number 200 Debounce delay for search filter (ms)
disabled boolean false Disable the select component
invalid boolean false Show invalid/error state

Properties (i-fc-select)

Property Type Description
label string Label text
required boolean Show required indicator (*)
errorMessage IFormControlErrorMessage Custom error messages configuration
formControlName string Reactive forms control name

Events

Event Type Description
onChanged EventEmitter<ISelectChange<T>> Emitted when selection changes
onOptionSelected EventEmitter<ISelectChange<T>> Emitted when user selects an option

Panel Positions

'top' | 'bottom' | 'left' | 'right' | 'top left' | 'top right' | 'bottom left' | 'bottom right'
