Button

i-button

Button uses i-button provide a clickable element, which can be used in forms, or anywhere that needs simple, standard button functionality. They may display text, icons, or both. Buttons can be styled with several attributes to look a specific way.


Properties :

Property Description Required Default Value Remark
disabled boolean Optional false
type IButtonType Optional 'button' 'button', 'submit', 'reset'
variant IButtonVariant Optional 'primary' 'primary', 'warning', 'danger', 'success', 'outline'
size IButtonSize Optional 'md' '2xs', 'xs', 'sm', 'md', 'lg'
icon string Optional '' See IIcon for available icons
loading boolean Optional false
loadingText string Optional 'Loading'

Event :

Event Description Required Default Value
onClick Function Optional (value: MouseEvent) => void

IButtonVariant

There are 5 button variants :

  • primary
  • success
  • warning
  • danger
  • outline

IButtonType

There are 3 button types :

  • button
  • submit
  • reset

IButtonSize

There are 5 button sizes :

  • 2xs
  • xs
  • sm
  • md
  • lg

Icons

See Icons section for available icons
