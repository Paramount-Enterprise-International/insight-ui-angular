Scroll

Utilities for enabling custom scroll behavior and styling scrollbars using CSS variables for overflow, thumb radius, and thumb color.


scroll

Class Styles
scroll --i-scroll-overflow-x: hidden;
--i-scroll-overflow-y: hidden;
--i-scroll-thumb-radius: var(--i-radius-full);
--i-scroll-thumb-color: var(--i-color-blue);
overflow-x: var(--i-scroll-overflow-x);
overflow-y: var(--i-scroll-overflow-y);
scroll-x --i-scroll-overflow-x: auto;
scroll-y --i-scroll-overflow-y: auto;
