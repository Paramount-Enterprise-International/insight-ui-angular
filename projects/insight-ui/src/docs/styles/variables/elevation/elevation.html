Elevation

Box shadow variables for creating consistent depth and elevation effects across the application. These variables provide predefined shadow styles for different elevation levels.


Elevation Variables Reference

Variable Name CSS Value Description
--i-elevation-sm 0px 1px 2px rgba(0, 0, 0, 0.05) Small Elevation: Subtle shadows for cards, buttons, and small interactive elements. Minimal depth for hover states and lightweight components.
--i-elevation-md 0px 4px 8px rgba(0, 0, 0, 0.08) Medium Elevation: Standard shadows for modal dialogs, dropdown menus, and card components. Suitable for medium-depth UI elements that need visual separation.
--i-elevation-lg 0px 8px 16px rgba(0, 0, 0, 0.12) Large Elevation: Prominent shadows for tooltips, popovers, and floating action buttons. Maximum depth for elements that should appear above all other content.
