Border Variables

CSS custom properties for consistent border styling across the application. Provides predefined border styles, colors, and semantic states.


Base Variables

Variable Default Value Description
--i-border-color var(--i-color-gray-200) Default border color
--i-border-radius 0px Default border radius for rounded corners
--i-border-style solid Default border style
--i-border-width 1px Default border width

Predefined Border Styles

Standard Borders

Variable Value Description
--i-border 1px solid var(--i-border-color) Default solid border
--i-border-dotted 1px dotted var(--i-border-color) Default dotted border
--i-border-subtle 1px solid var(--i-border-subtle-color) Subtle solid border
--i-border-subtle-dotted 1px dotted var(--i-border-subtle-color) Subtle dotted border
