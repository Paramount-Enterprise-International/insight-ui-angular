Icon Size

Icon size variables for creating consistent icon sizing across the application. These variables provide predefined sizes for icons at different scale levels.


Icon Size Variables Reference

Variable Name CSS Value Description
--fa-width auto Icon Width: Controls the width of icon elements. Set to 'auto' for proportional scaling based on icon height.
--i-icon-xs 10px Extra Small: For very small icons in tight spaces, toolbars, or inline with small text. Minimal visual presence.
--i-icon-sm 12px Small: Standard small icons for buttons, form controls, and list items. Suitable for most UI elements.
--i-icon-md 14px Medium: Default icon size for most applications. Used in navigation menus, card headers, and medium-sized buttons.
--i-icon-lg 16px Large: Prominent icons for page headers, primary actions, and important visual indicators.
--i-icon-xl 28px Extra Large: Feature icons, section headers, and standalone visual elements. High visibility.
--i-icon-2xl 32px 2X Large: Hero icons, empty states, and primary feature illustrations. Maximum impact.
--i-icon-3xl 64px 3X Large: Decorative icons, loading states, and large visual elements. Used sparingly for emphasis.
--i-icon-4xl 96px 4X Large: Landing page icons, marketing illustrations, and maximum-scale visual elements. Used only for emphasis.
