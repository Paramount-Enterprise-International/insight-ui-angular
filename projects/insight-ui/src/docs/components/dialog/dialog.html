Dialog

i-dialog

i-dialog is a reusable dialog/modal component for Angular applications. It supports custom content, actions, and various configurations with a flexible API.


Dialog Configuration (IDialogConfig)

Configuration options when opening a dialog via IDialogService.open():

Property Type Required Default Value Description
id string Optional Auto-generated Unique identifier for the dialog instance
data any Optional undefined Data passed to dialog component via I_DIALOG_DATA
width string Optional 'auto' Dialog width (e.g., '500px', '80%')
height string Optional 'auto' Dialog height (e.g., '400px', '60vh')
disableClose boolean Optional false If true, users cannot close with ESC key
backdropClose boolean Optional true If true, users can close by clicking backdrop

Events

Event Type Description
onOk EventEmitter<any> Emitted when the OK button is clicked
onConfirm EventEmitter<any> Emitted when the Confirm button is clicked
onSave EventEmitter<any> Emitted when the Save button is clicked
onCustomAction EventEmitter<IDialogActionObject> Emitted when a custom action button is clicked

DialogRef Methods

Method Return Type Description
afterClosed() Observable<any> Returns an observable that emits when dialog is closed
close(result?: any) void Programmatically closes the dialog with optional result

DialogAction Types

There are 5 predefined action types:

  • 'cancel' - Cancel button
  • 'save' - Save button
  • 'ok' - OK button
  • 'confirm' - Confirm button
  • 'custom' - Custom button with additional configuration

Custom action configuration includes:

  • label: string - Button text
  • variant: IButtonVariant - Button style variant
  • icon: string - Icon name
  • onClick: () => void - Custom click handler
  • className: string - Additional CSS classes
  • visible: boolean - Visibility toggle

IButtonVariant

There are 5 button variants :

  • primary
  • success
  • warning
  • danger
  • outline

Directives

i-dialog-close

Directive for creating close buttons inside dialog content:

Attribute Type Description
i-dialog-close string (optional) Value returned when dialog is closed
[iDialogClose] any Binding for result value

Service Methods

Method Parameters Return Type Description
open() component: Type<TComponent>, config?: IDialogConfig IDialogRef<TResult> Opens a new dialog with specified component
closeById() id: string, result?: any void Closes dialog by ID
closeAll() None void Closes all active dialogs
dialogs$ None Observable<IDialogInstance[]> Stream tracking all active dialogs
