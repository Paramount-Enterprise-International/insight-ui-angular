Border Radius

Border radius variables for creating consistent rounded corners across the application. These variables provide predefined radius values for different curvature levels.


Border Radius Variables Reference

Variable Name CSS Value Description
--i-radius-none none No Radius: Sharp corners.
--i-radius-2xs 5px Extra Extra Small: Subtle rounding for small elements.
--i-radius-xs 6px Extra Small: Light rounding.
--i-radius-sm 7px Small: Standard small rounding.
--i-radius-md 8px Medium: Default border.
--i-radius-lg 10px Large: Prominent rounding.
--i-radius-xl 16px Extra Large: Significant curvature.
--i-radius-2xl 24px 2X Large: Dramatic rounding.
--i-radius-full 9999px Full Circle: Creates perfect circles or pill-shaped elements.
