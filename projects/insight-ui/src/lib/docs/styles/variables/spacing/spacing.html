Spacing

Spacing variables for creating consistent layout and spacing across the application. These variables provide a systematic scale for margins, paddings, and gaps between elements.


Spacing Variables Reference

Variable Name CSS Value Usage & Description
--i-spacing-6xs 2px 6X Small: Minimal spacing for tight layouts. Used for subtle gaps between tightly packed elements or micro-adjustments.
--i-spacing-5xs 4px 5X Small: Very small spacing for compact interfaces. Suitable for icon-text spacing, small indents, or minimal padding.
--i-spacing-4xs 6px 4X Small: Compact spacing for dense components. Used for form field spacing, list item padding, or small element gaps.
--i-spacing-3xs 8px 3X Small: Standard small spacing. Common for button padding, input padding, and small container margins.
--i-spacing-2xs 10px 2X Small: Balanced small spacing. Used for card padding, section spacing, and consistent small gaps.
--i-spacing-xs 12px Extra Small: Versatile spacing for various components. Suitable for modal padding, dropdown spacing, and medium gaps.
--i-spacing-sm 16px Small: Default small spacing for most layouts. Used for paragraph spacing, content blocks, and comfortable padding.
--i-spacing-md 24px Medium: Standard spacing for content separation. Common for section margins, card spacing, and page layout gaps.
--i-spacing-lg 32px Large: Generous spacing for visual breathing room. Used for major section breaks, large containers, and emphasized separation.
--i-spacing-xl 48px Extra Large: Significant spacing for content hierarchy. Suitable for page sections, hero components, and prominent dividers.
--i-spacing-2xl 64px 2X Large: Maximum spacing for dramatic separation. Used for landing page sections, full-width banners, and major content breaks.
