Typography Variables

CSS custom properties for consistent typography styling across the application. Provides predefined font families, sizes, weights, spacing, and semantic text colors.


Font Family Variables

Variable Default Value Description
--i-font-heading 'Poppins', ui-sans-serif, system-ui, sans-serif Primary font for all heading elements (h1-h6)
--i-font-body 'Source Sans 3', ui-sans-serif, system-ui, sans-serif Primary font for body text and paragraph content
--i-font-sans ui-sans-serif, system-ui, -apple-system, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, 'Noto Sans', sans-serif Comprehensive system font stack for interface elements
--i-font-mono ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, 'Liberation Mono', 'Courier New', monospace Font stack for code, technical content, and data displays
