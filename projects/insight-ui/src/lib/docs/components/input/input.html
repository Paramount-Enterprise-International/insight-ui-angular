i-input, i-fc-input

Textbox component, i-input is a wrapper for input component, with several added functionality, i-fc-input is a wrapper of i-input that includes label and error message

IInputAddons

You can insert certain elements as an addon for i-input, these addon inserted through attributes prepend and append, the object that being inserted called IInputAddon, IInputAddon can be one of the following type : 

  • IInputAddonButton
  • IInputAddonLink
  • IInputAddonIcon
  • IInputAddonText
  • IInputAddonLoading

IInputMaskDirective

Reusable input mask directive, for now it only support date and time

Properties :

Property Description Required Default Value
mask IInputMask Required

IInputMask

An object that passed as IInputMaskDirective mask parameter

Properties :

Property Description Required Default Value
type IInputMaskType Required 'date', 'time'
format string Required
  • 'date' → e.g. 'dd/MM/yyyy', 'yyyy-MM-dd'
  • 'time' → e.g. 'HH:mm', 'HH:mm:ss'
