Pill Component

i-pill Component

The Pill component is a small, rounded UI element used for displaying tags, labels, or status indicators. It supports multiple sizes, variants, and can include a close button for removable items.


Component Overview

The Pill component is a standalone Angular component with the following features:

  • Selector: i-pill
  • Standalone: Yes
  • Change Detection: OnPush strategy for better performance
  • Content Projection: Uses <ng-content> for flexible content

Type Definitions

IPillSize

Value Description
'2xs' Extra extra small size
'xs' Extra small size
'sm' Small size
'md' Medium size (default)
'lg' Large size

IPillVariant

Value Description
'default' Default styling (default)
'primary' Primary color variant
'success' Success/green color variant
'warning' Warning/yellow color variant
'danger' Danger/red color variant

Properties :

Property Type Required Default Value Description
size IPillSize Optional 'md' Controls the size of the pill
variant IPillVariant Optional 'default' Controls the color variant of the pill
disabled boolean Optional false Disables the pill interaction and applies disabled styling
closable boolean Optional false Shows a close button for removable pills

Events :

Property Type Description
onClose EventEmitter<MouseEvent> Emitted when the close button is clicked (only when closable is true)
onClick EventEmitter<MouseEvent> Emitted when the pill is clicked (excluding close button clicks)
