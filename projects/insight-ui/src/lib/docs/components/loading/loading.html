Loading

i-loading

i-loading is a loading indicator component that displays a spinner with optional text. It provides visual feedback during asynchronous operations or content loading.


Properties

Property Type Required Default Value Description
label string Optional 'Loading..' Text displayed next to the loading spinner
light boolean Optional false If true, displays a light-colored spinner (for dark backgrounds)

CSS Classes

The component applies the following CSS classes:

Class Applied When Description
spinner-border Always Base class for the spinner animation
spinner-border-sm Always Small size modifier for the spinner
light [light]="true" Light color variant for dark backgrounds

Host Binding

The component uses @HostBinding to add the light attribute to the host element when [light]="true".
