# Datepicker Component Documentation

Datepicker

i-datepicker & i-fc-datepicker

The Datepicker component system provides a comprehensive date selection solution with two main components: i-datepicker for core date selection and i-fc-datepicker for form-controlled usage with label and error handling. Both components support Angular Forms integration via ControlValueAccessor.


Components Overview

The Datepicker system consists of these components:

  • i-datepicker - Core datepicker component with calendar popup
  • i-fc-datepicker - Form-controlled wrapper with label and error handling

Component Properties

i-datepicker Properties

Property Type Required Default Value Description
placeholder string Optional '' Input placeholder text
disabled boolean Optional false Disables the datepicker interaction
invalid boolean Optional false Visual invalid state
format string Optional 'dd/MM/yyyy' Date display format (supports yyyy, MM, dd tokens)
panelPosition IDatepickerPanelPosition Optional 'bottom left' Calendar popup position relative to input
value Date | string | null Optional null Current date value (accepts Date object or string)

i-fc-datepicker Properties

Property Type Required Default Value Description
label string Optional '' Form control label
placeholder string Optional '' Input placeholder text
format string Optional 'dd/MM/yyyy' Date display format
panelPosition IDatepickerPanelPosition Optional 'bottom left' Calendar popup position
errorMessage IFormControlErrorMessage Optional undefined Custom error messages for validation
value Date | null Optional null Current date value

Events

i-datepicker Events

Event Type Description
onChanged EventEmitter<Date | null> Emitted when date value changes

Type Definitions

IDatepickerPanelPosition

Value Description
'top' Calendar appears above input
'bottom' Calendar appears below input
'left' Calendar appears to the left of input
'right' Calendar appears to the right of input
'top left' Calendar appears above and aligned left
'top right' Calendar appears above and aligned right
'bottom left' Calendar appears below and aligned left
'bottom right' Calendar appears below and aligned right

Date Format Tokens

Token Description Example
yyyy 4-digit year 2024
MM 2-digit month (01-12) 01, 12
dd 2-digit day (01-31) 01, 31
